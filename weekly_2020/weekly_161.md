# 实用工具

1. [Monotty Desktopio](https://github.com/netxs-group/vtm): 终端内基于文本的桌面环境
2. [Pydrawing](https://github.com/CharlesPikachu/pydrawing): 图片/视频美化工具包
3. [Emoji Semantic Search](https://github.com/lilianweng/emoji-semantic-search): 表情图标自然语言语义搜索引擎
4. [98五笔输入法](http://www.98wubi.com/): 基于 Rime 的开源五笔输入法
5. [bat](https://github.com/sharkdp/bat)：一个cat命令的替代品，对大部分编程语言和标记语言提供语法高亮，能从 git 中获取文件的修改并展示在边栏
6. [Harmonoid](https://github.com/harmonoid/harmonoid): GitHub 上一款优雅、简洁的开源音乐播放器
7. [vCards CN](https://github.com/metowolf/vCards): 国内开发者在GitHub开源的一个项目可为常用联系人导入头像，优化 iOS/Android 来电、信息界面体验
8. [forsaken-mail](https://github.com/malaohu/forsaken-mail): 即收即毁的临时邮件服务
9. [BluetoothLE Battery Monitor](https://github.com/MUedsa/BluetoothLEBatteryMonitor): 开源的低功耗蓝牙设备电量显示工具
